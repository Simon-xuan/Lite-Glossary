# Lite-Glossary (轻量级词汇表)

**Lite-Glossary** 是一款为 WordPress 打造的极简术语工具提示插件。它旨在替代臃肿的同类插件，通过原生技术栈实现高效、简洁的术语高亮与释义展示。

---

## ✨ 功能特点

* **专用管理面板**：通过自定义文章类型（CPT）轻松管理所有专业术语。
* **智能匹配算法**：自动识别内容术语，智能避开已有的超链接 (`<a>`) 和标题标签 (`<h1>`-`<h6>`)，防止布局破坏。
* **首词高亮模式**：支持全局设置，仅对文章内首次出现的术语进行高亮，保持页面整洁。
* **极速原生前端**：采用 **纯 CSS + 原生 JavaScript (Vanilla JS)**，零依赖（无 jQuery），不加载冗余库。
* **批量便捷导入**：支持通过 CSV 文本格式快速导入大量术语数据。
* **高性能架构**：内置缓存机制，大幅减少数据库查询压力。
* **原生中文支持**：原生使用中文，无理解问题。
* **数据互通**：与CM Tooltip数据互通（其他插件数据互通功能请自行尝试，不做保证）

## 🚀 安装方法

1.  **下载**：从 [Releases](../../releases) 页面获取最新的 `LiteGlossary.zip`。
2.  **上传安装**：
    * 登录 WordPress 后台 -> **插件** -> **添加新插件** -> **上传插件**。
    * 选择 `LiteGlossary.zip` 文件并安装。
3.  **激活**：安装完成后点击 **激活插件**。

---

## 📖 使用指南

### 1. 单个添加术语
* 进入后台 **词汇表** -> **添加新术语**。
* **标题栏**：输入术语名称。支持别名，使用 `|` 隔开。
* **格式示例**：
    * `Phone,手机`：当文章出现 "Phone" 时显示 "手机"。
    * `TV|Television,电视`：匹配 "TV" 或 "Television" 时均显示 "电视"。

### 2. 批量导入术语
在插件设置页面的导入框中，按照以下格式输入：
```text
术语1|别名,释义
术语2,释义
...
```

❓ 常见问题 (FAQ)
Q：支持中文术语吗？ A：完美支持。插件在正则匹配和数据库存储上均对中文等字符做了优化。

Q：如何自定义工具提示的外观？ A：您可以直接修改插件目录下的 assets/css/tooltip.css。

Q：批量导入失败怎么办？ A：请检查逗号是否为英文半角 ,，并确保每一行符合 名称,释义 的规范。
